简单题总述 ​

在“计算机系统”方向的简单题部分，你需要做以下的准备：

  • 学习过“日常基础”的Linux操作部分
  • 在个人电脑上拥有Linux系统（wsl、虚拟机等）
  • 在Linux系统上拥有ide软件（vim、vscode，这不是必要的，但是入门时使用这类软件可以节省较多时间）
  • 对C语言有过了解或者抱有较高的兴趣
  • 对于题干中加粗的概念，我们希望你都可以进行了解学习并掌握

Step 1. My name is XXX ​

创建一个新的.c后缀文件，输入以下内容

c
#include <stdio.h>

int main(){
  printf("My name is $姓名$.\n");
  printf("I am $年龄$ years old.");
  return 0;
}
#include <stdio.h>

int main(){
  printf("My name is $姓名$.\n");
  printf("I am $年龄$ years old.");
  return 0;
}

$姓名$ 替换为你想填入的姓名即可, $年龄$ 替换为你想填入的数字。

进行合适的替换后，尝试在你的IDE中运行该程序，你将会得到类似于以下内容的结果：

shell
My name is 秦始皇.
I am 2284 years old.
My name is 秦始皇.
I am 2284 years old.

IDE的选择首推VS Code（并非一定），安装与相关配置请自行上网搜索。

恭喜！现在你已经完成了一个C语言程序，可是对于其具体含义的理解可能还有待学习。现在请你自主进行C语言的基础语法学习，尝试回答下列问题来进行一定的基础检验：

  1. 解释一下#include <stdio.h>的含义，分析在上述程序中它的作用。
  2. main函数在整个程序中起到什么作用？为什么main函数要定义为int类型？
  3. int类型是计算机存储什么元素的方式？了解一下C语言中的基本数据类型。
  4. 转义字符\n的含义是什么？了解一下常见的转义字符及其含义，回答回车键本质上是什么。

Step 2. 动态交互 ​

我们刚刚完成的程序已经可以很好的打印我们想要姓名与年龄，可是，小明敏锐地发现了一个问题：如果我们想进行一定的更改，都要直接修改源代码。通过搜索引擎，他得知这样不仅太过繁琐，而且不利于我们对程序进行测试，在实际应用中也几乎不可行。所以小明打算对程序进行一定的改写，使之可以实现动态交互。于是小明打算进一步学习变量、数组、流程控制 - 循环结构、流程控制 - 逻辑表达式等知识。但是任何新知识的学习都不是一蹴而就的，必然要经历疑问与解答的环节，小明也不例外，针对上述内容，他提出了以下问题，请你解答他的疑惑。

  1. 变量类型： 什么是变量的类型？它为什么重要？如果你想存放你的年龄（比如 25），应该选择哪种基本变量类型（如整数、浮点数）？如果你想存放单词 "apple"，使用一个char类型的字符变量能实现吗？尝试编程验证你的想法，并思考正确的存储方式应该是什么？
  2. 数组的起始与边界： 数组的第一个元素下标通常是从 0 还是 1 开始？访问数组元素时，下标超出其有效范围（例如，长度为 5 的数组访问 arr[5] 或 arr[-1]）会导致什么问题？为什么数组越界是一个常见的、危险的错误？
  3. 流程控制 - 循环结构： 循环结构（如 for, while）是如何控制代码块重复执行的？请分别简述它们的基本结构。for 循环的初始化、条件判断和迭代部分各自的作用是什么？while 循环和 do...while 循环在首次执行条件判断上有何关键区别？尝试用两种不同的循环结构实现同一个任务（例如，计算 1 到 10 的和），体会它们的异同。
  4. 流程控制 - 逻辑表达式：在条件判断（如 if, else if）和循环条件中使用的逻辑表达式（如age > 18 && score >= 60）与算术表达式（如a + b * c）有何本质区别(尝试从运算对象和运算结果分析)？运算逻辑符&&,||,!的含义分别是什么？尝试编写包含多个条件的逻辑表达式并预测其输出结果。

听取了你的回答，小明恍然大悟，尝试对程序进行了修改，以期可以实现如下功能：

  1. 设置合适的变量用于存储输入的姓名与年龄。

  2. 利用设置的变量，以原格式打印姓名与年龄。

  3. 在每一次的读取输入->打印内容结束后，记录进行过读取输入->打印内容操作的次数，打印提示询问用户是否继续输入。

  4. 若判断得到用户需要继续输入，则重复之前的读取输入->打印内容操作。

  5. 若判断得到用户不需要继续，则打印记录的进行的读取输入->打印内容操作次数，然后结束程序。

Step 3. 自定义函数 ​

函数是执行特定任务的自包含代码块，是程序的基本构建单元。在C语言中，函数分为库函数和自定义函数。库函数前面已经有所触及，这里就不赘述了。而自定义函数，就是在实际代码编写中，程序员为了实现库函数无法满足的需求时，自主创建的函数。函数的一般结构如下：

返回类型 函数名(参数列表) {
    函数体
}
返回类型 函数名(参数列表) {
    函数体
}

我们可以举一个比较直观的例子：

假设我们的原代码如下：

c
#include <stdio.h>

int main(){
  int a, b;
  scanf("%d %d", &a, &b);
  printf("%d", a + b);
}
#include <stdio.h>

int main(){
  int a, b;
  scanf("%d %d", &a, &b);
  printf("%d", a + b);
}

为了让代码更清晰、更容易重复使用计算两数之和的功能，我们可以把 a + b 这个计算过程单独定义成一个函数。这个过程可以称之为将a + b封装成函数int add(int a, int b), 改写后代码如下：

c
#include <stdio.h>

int add(int a, int b){
  return a + b;
}

int main(){
  int a, b;
  scanf("%d %d", &a, &b);
  printf("%d", add(a,b));
}
#include <stdio.h>

int add(int a, int b){
  return a + b;
}

int main(){
  int a, b;
  scanf("%d %d", &a, &b);
  printf("%d", add(a,b));
}

这就是自定义函数的一个简单实现。当我们运行程序时，程序从主入口（在C语言中，程序的主入口一般情况下都是main 函数）开始执行，遇到自定义函数(如add(a,b))调用时：

  1. 暂停当前执行，保存上下文，传递参数；
  2. 跳转至函数体内按序执行（可嵌套调用其他函数）；
  3. 执行完毕后，返回结果（若有），恢复上下文；
  4. 从调用处下一条指令继续执行。

在这里，我们封装的函数对当前代码整体编写的效果并不明显，也就是说，我们在这里定义的add()函数，针对当下的程序，其实并不是必要的。那么什么时候封装函数是必要的呢？

在程序设计中，设置函数的核心目的有两点：代码复用和逻辑组织

代码复用是指通过抽象化特定功能模块为独立单元，实现算法逻辑的"一次定义，多次调用"，从而消除冗余代码并确保功能一致性。简单来说，就是把常用的操作打包成工具，随用随取。

逻辑组织是指通过函数边界建立抽象层次结构，将复杂系统分解为高内聚、低耦合的功能模块，实现认知负荷的有效管理。简单来说，就是把代码按功能分装进盒子（函数），便于后续查找、维护和修改。

下面我们设置一个情景，方便你更好地了解。

小明、小强、小林是某人三项（共有A、B、C三项）的业余爱好者，某次，三人一同参加了某人三项的趣味赛，成绩如下（满分100）：

小明：77 85 91

小强：82 84 84

小林：72 83 100

请你通过平均分和方差的计算，综合考虑后，为三人综合成绩由高到低排序(成绩相等时，对顺序不做要求)。综合考虑的具体方法为：综合成绩 = 3 * 平均分 - 方差 / 3;

下面给出仅由一个main()函数完成的程序：

c
#include <stdio.h>

int main(){
  int x1, x2, x3;
  int y1, y2, y3;
  int z1, z2, z3;

  printf("请输入小明的三项成绩（顺序为A B C,以一个空格为间隔）：");
  scanf("%d %d %d", &x1, &x2, &x3);
  printf("请输入小强的三项成绩（顺序为A B C,以一个空格为间隔）：");
  scanf("%d %d %d", &y1, &y2, &y3);
  printf("请输入小林的三项成绩（顺序为A B C,以一个空格为间隔）：");
  scanf("%d %d %d", &z1, &z2, &z3);

  int p1 = (x1 + x2 + x3) / 3;
  int f1 = ((p1 - x1) * (p1 - x1) + (p1 - x2) * (p1 - x2) + (p1 - x3) * (p1 - x3)) / 3;
  int zh1 = 3 * p1 - f1 / 3;
  
  int p2 = (y1 + y2 + y3) / 3;
  int f2 = ((p2 - y1) * (p2 - y1) + (p2 - y2) * (p2 - y2) + (p2 - y3) * (p2 - y3)) / 3;
  int zh2 = 3 * p2 - f2 / 3;

  int p3 = (z1 + z2 + z3) / 3;
  int f3 = ((p3 - z1) * (p3 - z1) + (p3 - z2) * (p3 - z2) + (p3 - z3) * (p3 - z3)) / 3;
  int zh3 = 3 * p3 - f3 / 3;
  
  if (zh1 >= zh2 && zh2 >= zh3) {
      printf("小明 > 小强 > 小林");
  } else if (zh1 >= zh3 && zh3 >= zh2) {
      printf("小明 > 小林 > 小强");
  } else if (zh2 >= zh1 && zh1 >= zh3) {
      printf("小强 > 小明 > 小林");
  } else if (zh2 >= zh3 && zh3 >= zh1) {
      printf("小强 > 小林 > 小明");
  } else if (zh3 >= zh1 && zh1 >= zh2) {
      printf("小林 > 小明 > 小强");
  } else { // zh3 >= zh2 && zh2 >= zh1
      printf("小林 > 小强 > 小明");
  }

  return 0;
}
